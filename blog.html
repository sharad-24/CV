How to get start with Ardino software

Hi guys welcome to my second blog. In this blog we are talking about Arduino software which is commonly used for the programing of radio control toys like drones, RC cars, helicopters etc.

Arduino is an open-source electronics platform based on easy-to-use hardware and software. Arduino boards are able to read inputs light on a sensor, a finger on a button, or a twitter message and turn it into an output activating of a message in kind of command for another hardware to perform desirable task.

In markets different type of Arduino board are available like:

• Arduino Uno (R3)

• Lily Pad Arduino Board.

• Red Board Arduino Board.

• Arduino Mega (R3) Board.

• Arduino Leonardo Board.

• Arduino Shields.

For beginners Arduino Uno is good for accomplish their code on it.

Steps to be followed to get a quick start with Arduino:

1: Install Arduino software from https://www.arduino.cc/en/main/software

2: Now download multiwii software (https://code.google.com/archive/p/multiwii/) for simulating your Arduino software on computer itself

3: The other advantages of multiwii is that, you found all the coding of different instruments and toys already done, so you only have to export them in your Arduino and enjoy its functioning in Arduino

I hope this block help you to begin with arduino and multiwii. Thank you
